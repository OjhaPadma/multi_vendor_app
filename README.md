# multi_vendor_app
Multi-Vendor Food Ordering App (Flutter)  Flutter-based multi-vendor food ordering app where multiple restaurants can list menus and manage orders. Users can browse restaurants, add items to cart, place orders, and track deliveries. Built with clean architecture, REST APIs, and Riverpod state management.

🍔 Multi-Vendor Food Ordering App (Flutter)

A modern multi-vendor food ordering mobile application built with Flutter that allows multiple restaurants to manage their menus and receive orders from customers through a single platform.

The application demonstrates clean architecture, scalable state management, and a smooth user experience, making it suitable for real-world marketplace applications.
